## 목적 및 목표

잠재 고객(경남 소상공인·중소기업 대표)이 검색이나 링크를 통해 처음 방문했을 때, 27년 경력의 신뢰도를 전달하고 카카오톡 무료 상담으로 전환시키는 랜딩페이지.

**성공 기준:** 방문자가 페이지를 스크롤한 후 카카오톡 상담 버튼을 클릭한다.

- **Vanilla JavaScript** — 모바일 메뉴, 스크롤 페이드인 애니메이션
- **배포:** 서버 불필요 (GitHub Pages / Netlify 등 정적 호스팅)

## 페이지 구조

### 1. 네비게이션 (fixed top)
- 좌: "호짱쌤" 로고 (골드 텍스트)
- 우: "무료 상담" CTA 버튼 (골드 배경, 네이비 텍스트)
- 모바일: 햄버거 메뉴 → 드롭다운 섹션 링크 (소개, 서비스, 연락처)
- 스크롤 시 배경 블러 효과

### 2. Hero 섹션
- 배경: 네이비 + 대각선 골드 포인트 장식
- H1: "27년 경력 정책자금 전문가"
- H2: "경남 소상공인·중소기업의 든든한 파트너"
- 소개 한 줄: "1999년부터 여신심사역으로 시작한 실전 컨설턴트"
- CTA 버튼: "카카오톡으로 무료 상담받기 →" (골드, 큰 버튼)
- 카카오톡 채널 링크 연결 (URL은 추후 교체 가능한 placeholder `#kakao`)

### 3. 신뢰지표 섹션
- 흰 배경, 3열 그리드 (모바일: 1열)
- 숫자 강조 카드:
  - **27년** — 전문 컨설팅 경력
  - **1999년** — 금융기관 여신심사역으로 출발
  - **경남 전문** — 소상공인·중소기업 특화

### 4. 소개 섹션
- 라이트그레이 배경
- 좌: 프로필 이미지 placeholder (원형, 200×200)
- 우: 소개 텍스트 (제공된 소개문 그대로 사용)
- 경력 타임라인: 1999년(여신심사역) → 현재(정책자금 컨설턴트)

### 5. 서비스 섹션
- 흰 배경, 섹션 제목: "주요 서비스"
- 4개 카드 그리드 (모바일: 2열, 데스크탑: 4열)

| 카드 | 이모지 | 제목 | 설명 |
|------|--------|------|------|

| 3 | 🏆 | 기업인증 지원 | 벤처·이노비즈 등 인증 컨설팅 |
